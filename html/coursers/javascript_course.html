Lesson: CSS Basics

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls the design elements like color, font, and spacing.

Example:


h1 {
    color: blue;
    text-align: center;
}
p {
    font-size: 16px;
    color: green;
}

In this example, the heading will be centered and colored blue, while the paragraph will be green and have a font size of 16px.
